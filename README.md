## 🚀 3つの使い方

### 方法A: PCブラウザで使う（一番簡単）

### 方法B: スマホで使う（AirDrop経由）
1. Finderで `stock-app` フォルダを選択
2. 右クリック → 共有 → AirDrop → iPhone送信

5. ホーム画面にアプリ風アイコン出現 🎉

### 方法C: 世界中どこからでも使う（Netlify Drop）
1. https://app.netlify.com/drop を開く（アカウント不要）
2. `stock-app` フォルダを **ドラッグ&ドロップ**
3. URL発行（例: `happy-nisa-1234.netlify.app`）
4. iPhone・Android・どの端末でもそのURLで開ける
5. 開いたブラウザで「ホーム画面に追加」

**✅ Netlify Drop は完全無料 · アカウント不要 · 30秒で完了**

## 💾 データ同期（端末間）

**重要: LocalStorage はブラウザごとに独立**
PCで入力したデータはスマホには自動同期されません。以下の方法で手動同期してください。

### 手順
1. **PC側**: ☰メニュー → 「📤 エクスポート」ボタン → JSONダウンロード
2. **AirDrop/メール/LINE/iCloud** でスマホに送信
3. **スマホ側**: ☰メニュー → 「📥 インポート」→ ファイル選択
4. 完全復元完了 ✅

### エクスポートされるデータ
- 保有銘柄・株数・取得単価
- NISA管理情報
- AIチャット履歴
- アドバイザー履歴
- 取引ルール設定
- トレード日誌
- 誕生日（18歳カウントダウン用）
- 投資メモ

## ⚠️ 注意事項

1. **株価の自動更新なし**（日本株APIのCORS制約のため）
   - TradingViewチャートは15-20分遅延で表示
   - 発注時はSBI証券アプリで最終確認必須

2. **データはローカル保存のみ**
   - ブラウザのキャッシュクリアで消える
   - シークレットモード利用不可
   - 定期的にエクスポートでバックアップ推奨

3. **Service Worker（オフライン機能）** は`file://` では動作しません
   - Netlify/GitHub Pages 経由で開いた場合のみ有効

4. **投資は自己責任**
   - 全ての分析は参考情報
   - 将来の株価上昇を保証するものではありません

## 🔧 トラブルシューティング

### Q. スマホで開くとレイアウトが崩れる
A. 画面幅900px以下で自動的にモバイル用レイアウト切替。キャッシュクリアで解決する場合あり。

### Q. TradingViewチャートが表示されない
A. 広告ブロッカー無効化 or 各カードの「↗」リンクで新規タブで開く

### Q. データが消えた
A. ブラウザのキャッシュクリア・シークレットモードで消えます。エクスポート済みJSONからインポートで復元。

### Q. iOSでホーム画面追加してもSafariバーが表示される
A. 一度「ホーム画面に追加」で作ったアイコンから起動し直してください。旧アイコンは削除。

### Q. アップデート時にデータは消える？
A. 消えません。LocalStorageは残ります。ただし重大アップデート前にエクスポートを推奨。
